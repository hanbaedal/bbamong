# PPAMONG 야구 예측 게임 — 방법 요약

## 게임이란?

실시간 야구 경기를 보면서 **「다음 타석 결과」**를 맞추는 예측 게임입니다.  
맞추면 **포인트(참여 기록)**를 얻고, 틀리면 베팅한 포인트를 잃습니다.

---

## 참여 방법 (회원)

1. **로그인** (`/login`) — 일반·소셜·게스트 로그인 가능
2. **경기 참여** (`/prediction`) — 하단 메뉴 또는 홈에서 진입
3. **오늘 경기 선택**
4. 매니저가 **예측 시작**을 열면 **5가지 중 하나** 선택
5. 확인 후 **100 포인트** 베팅
6. 결과 발표 후 승·패에 따라 포인트 정산

---

## 예측 선택지 (5가지)

| 선택 | 의미 |
|------|------|
| **1루** | 1루타 |
| **2루** | 2루타 |
| **3루** | 3루타 |
| **홈런** | 홈런 |
| **아웃** | 아웃 |

---

## 한 라운드 흐름

```
매니저: 예측 라운드 시작
    ↓
회원: 5가지 중 하나 선택 + 100P 베팅
    ↓
매니저: 실제 타석 결과 입력 → 예측 종료
    ↓
맞춘 사람: 패자들이 건 포인트를 나눠 받음
틀린 사람: 베팅 포인트 차감
    ↓
다음 이닝/라운드 반복 (경기 종료까지)
```

### 규칙 요약

- **라운드당 1번만** 예측 가능 (같은 라운드에서 선택만 바꿀 수 있음, 결과 확정 전까지)
- 예측은 **매니저가 연 예측 시간**에만 가능
- 경기 **취소·종료** 후에는 예측 불가

---

## 포인트

| 항목 | 내용 |
|------|------|
| **베팅** | 예측 1회당 **100P** (기본) |
| **승리** | 틀린 사람들이 건 포인트를 **맞춘 사람 수**로 나눠 지급 |
| **패배** | 베팅 100P 차감 |
| **기부** | 승리 후 상금 일부를 **기부**할 수 있음 |
| **적립** | 출석, 광고 시청, 회원가입 보너스 등으로도 포인트 획득 |

보유 포인트는 앱 화면에서 **「보유 참여 기록」**으로 표시됩니다.

---

## 실시간 연동

- **WebSocket**으로 예측 시작·종료·결과를 실시간 수신
- 매니저 앱에서 라운드를 열고 결과를 넣으면, 회원 앱에 바로 반영

---

## 역할별 역할

| 역할 | 하는 일 |
|------|---------|
| **회원 (User 앱)** | 경기 선택 → 예측 → 포인트 획득·사용 |
| **매니저 (Manager 앱)** | 경기 진행, 예측 라운드 시작/종료, 결과 입력 |
| **관리자 (Admin)** | 경기·회원·포인트·광고 등 전체 운영 |

---

## 앱 구조 (참고)

| 구분 | 경로 | 설명 |
|------|------|------|
| **게임** | `/prediction` | 예측 참여 |
| **빠몽이의 보물창고** | `ppamong.com`, `/home` | 쇼핑·소개 (게임과 별도) |
| **게임 가이드** | `/home/game-guide` | 관리자가 등록한 상세 안내 |

- 상단 **빠몽이 로고**: 게임 ↔ 홈페이지(보물창고) 이동
- 게임 화면: 하단 메뉴(초대·출석·게시·추가·로그아웃)
- 보물창고: 게임용 하단 메뉴 없음, 로그인/로그아웃만 표시

---

## 매니저 경기 진행 순서 (운영 참고)

1. 담당 경기 선택
2. **경기 시작**
3. 이닝마다 **예측 라운드 시작**
4. 회원 예측 대기
5. **결과 입력** → 예측 종료 → 포인트 정산
6. 9이닝까지 반복
7. **경기 종료**
