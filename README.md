# bbamong (ppamong.com)

React + Express 기반 예측/매치 서비스. MongoDB Atlas + Redis.

## 로컬 실행

1. `.env.example`을 참고해 `.env` 작성
2. `npm install`
3. `npm run dev`

## 배포

Replit Autoscale 또는 Node 호스트에서 `npm run build` 후 `npm run start`.

자세한 체크리스트: [docs/PPAMONG_DEPLOY_CHECKLIST.md](docs/PPAMONG_DEPLOY_CHECKLIST.md)
